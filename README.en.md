# 人类自举自化工程

#### Description
人类自举自化工程：人当前是一份用DNA书写的运行在地球物理环境硬件下的碳基生命操作系统程序，人完全弄懂这份程序叫做自举，自举后自主决定进化方向叫做自化。人主要指的是大脑，其它都是输入输出设备，都是外设。大脑的本质是地球物理环境模拟器，语言文字是大脑间基于地球物理环境编码的信息交换协议。
